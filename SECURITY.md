# 安全策略

## 支持的版本

使用此部分告知用户哪些项目版本当前正在接收安全更新。

| 版本   | 支持状态            |
| ------ | ------------------- |
| 5.1.x  | :white_check_mark:  |
| 5.0.x  | :x:                 |
| 4.0.x  | :white_check_mark:  |
| < 4.0  | :x:                 |

## 报告漏洞

使用此部分告知用户如何报告漏洞。

请告知用户报告渠道、报告漏洞后预期多久能收到更新、漏洞被接受或拒绝后的处理流程等。
